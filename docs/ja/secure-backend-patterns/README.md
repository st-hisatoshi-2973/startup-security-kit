# セキュアバックエンドパターン

このディレクトリでは、バックエンド開発における実践的なセキュリティパターンをまとめています。

チェックリストが「何を確認するか」を示すのに対し、
本パターンでは **どのように設計・実装するか** に焦点を当てています。

---

# 目的

これらのパターンは、次のギャップを埋めることを目的としています。

* セキュリティの原則
* 実際の実装

各ドキュメントでは、特定のテーマについて

* シンプルな解説
* 実務で使える設計指針
* よくある問題

を整理しています。

---

# 含まれるパターン

本パターンは、バックエンドの設計レベルのセキュリティに焦点を当てています。

| パターン                  | 内容           |
| --------------------- | ------------ |
| jwt-authentication.md | JWT認証の設計パターン |

---

# 利用方法

これらのパターンは、次のような場面で利用することを想定しています。

* システム設計時
* バックエンド設計・実装・レビュー時
