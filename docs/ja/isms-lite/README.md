# ISMS Lite

ISMS Lite は、スタートアップや小規模チーム向けに設計された軽量な情報セキュリティマネジメントシステムです。

大企業向けフレームワークのような複雑さを避けつつ、小規模組織でも実践できるセキュリティガバナンスを提供することを目的としています。

---

# 目的

多くのセキュリティフレームワークは大企業向けに設計されており、小規模チームでは導入が難しい場合があります。

ISMS Lite は、次のような基本的なセキュリティ管理を実践できるようにすることを目的としています。

* 情報資産の把握
* セキュリティリスクの評価
* セキュリティポリシーの策定
* インシデント対応
* 継続的なセキュリティ改善

---

# PDCAサイクル

ISMS Lite は、簡略化した **PDCAサイクル** に基づいて運用されます。

* **Plan** — セキュリティポリシー策定とリスクアセスメント
* **Do** — セキュリティ対策の実装と運用
* **Check** — 内部監査による確認
* **Act** — 改善とセキュリティ対策の強化

# 利用方法

1. テンプレートを自社のドキュメントリポジトリにコピーします
2. 組織の環境に合わせて内容を調整します
3. 担当者や責任者を定義します
4. 定期的に見直しと更新を行います

---

# 注意

このディレクトリに含まれるドキュメントは **テンプレートおよび参考例** です。

実際の運用では、各組織の

* 業務内容
* システム構成
* セキュリティ要件

に合わせて調整する必要があります。

ISMS Lite は、小規模組織がセキュリティガバナンスを始めるための **出発点** として設計されています。
