# 情報資産台帳

## 目的

本ドキュメントは、組織が管理する情報資産を一覧化し、適切に管理することを目的とします。

情報資産を明確にすることで、リスク評価やセキュリティ対策を実施しやすくなります。

---

# 情報資産一覧

| 資産          | 管理者 | 保管場所   | 機密性 |
| ----------- | --- | ------ | --- |
| GitHubリポジトリ | 管理者 | GitHub | 高   |
| データベース      | 管理者 | クラウド   | 高   |
| ソースコード      | 管理者 | GitHub | 高   |
| アプリケーションログ  | 管理者 | クラウド   | 中   |

---

# 機密性レベル

機密性は以下の基準で分類します。

| レベル | 説明         |
| --- | ---------- |
| 高   | 顧客データや機密情報 |
| 中   | 業務データ      |
| 低   | 公開可能な情報    |

---

# 管理

情報資産は定期的に見直し、必要に応じて更新します。
