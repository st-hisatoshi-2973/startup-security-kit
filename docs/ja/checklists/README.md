# セキュリティチェックリスト

このディレクトリには、開発者向けの実践的なセキュリティチェックリストをまとめています。

このチェックリストは、開発プロセスの中で基本的なセキュリティ対策を実践できるようにすることを目的としています。

セキュリティポリシーやフレームワークとは異なり、**開発者が実際の開発作業の中で実行できる具体的な対策**に焦点を当てています。

---

# 含まれるチェックリスト

| チェックリスト                         | 内容                         |
| ------------------------------- | -------------------------- |
| developer-security-checklist.md | アプリケーション開発における基本的なセキュリティ対策 |

---

# 利用方法

このチェックリストは、開発やコードレビューの中で利用することを想定しています。

例：

* Pull Request のレビュー時に確認する
* 新しい機能を設計するときに確認する
* 開発ガイドラインの一部として利用する

---

# 対象範囲

このチェックリストでは、開発者レベルのセキュリティ対策に焦点を当てています。

* 認証
* 認可
* APIセキュリティ
* シークレット管理
* ログと監視
* 依存関係セキュリティ

このチェックリストは、**ISMS Lite のような組織レベルのセキュリティ方針を、開発者の実践に落とし込むこと**を目的としています。
