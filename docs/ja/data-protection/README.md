# データ保護

このディレクトリでは、機密情報や重要データをどのように保護するかについて扱います。

近年のシステムでは、機密情報は単に保存されるだけでなく、

* AI
* API
* ログ
* CI/CD
* 外部サービス

など、さまざまな境界を通過します。

そのため、

**「どこに保存するか」だけではなく、「どこへ送られるか」を制御すること**

が重要になっています。

---

# このディレクトリで扱う内容

このディレクトリでは、主に次のような観点を扱います。

* 機密情報をどのように検出するか
* 境界を越える前にどのように制御するか
* AIや外部サービス利用時にどう扱うか
* ログや監査とどのように連携するか
* 誤検知や見逃しをどのように考えるか

重要なのは、

**「漏洩後に探す」のではなく、「外へ出る前に制御する」**

という考え方です。

---

# AI Gatewayとの関係

AI利用では、ユーザー入力や内部データがそのまま外部のモデルAPIへ送信される可能性があります。

そのため、AI Gatewayでは、

* 何を送信するか
* 何を送信してはいけないか
* どこで制御するか

を整理する必要があります。

このディレクトリでは、特に

**「何を境界の外へ出さないか」**

という観点を扱います。
