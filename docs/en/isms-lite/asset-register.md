# Asset Register

## Purpose

This document lists the information assets managed by the organization.

Identifying assets helps the organization evaluate risks and apply appropriate security controls.

# Confidentiality Levels

Assets are classified according to the following confidentiality levels.

| Level  | Description                                 |
| ------ | ------------------------------------------- |
| High   | Sensitive information such as customer data |
| Medium | Internal operational data                   |
| Low    | Public or non-sensitive information         |

# Maintenance

The asset register should be reviewed periodically and updated when necessary.
